## 3. 실행 로드맵

```
Week 1: Affiliate 인프라 ────────────────────────
  ├─ accommodation.py에 affiliate ID 추가
  ├─ parser.py Step 2에 "실행 가이드" 섹션
  ├─ AdModule.tsx 실제 파트너 연결
  ├─ GA4 이벤트 추적 설정
  └─ affiliate_clicks DB 테이블

Week 2: Freemium Gate ───────────────────────────
  ├─ Stripe Checkout + 웹훅 (api/billing.py)
  ├─ users 테이블 subscription 컬럼
  ├─ /api/detail usage_count 미들웨어
  ├─ 프론트엔드 페이월 모달
  └─ 50명 소프트 런칭

Phase 3 (1-2개월 후): 큐레이션 킷 ──────────────
  ├─ 페르소나별 번들 구성
  └─ 파트너 협의

Phase 4 (3-6개월 후): 마켓플레이스/Autopilot ────
  ├─ 현지 서비스 매칭
  └─ 실시간 리밸런싱 엔진
```
