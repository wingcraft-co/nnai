# NomadNavigator AI - 핵심 기능 하이라이트

> 해커톤 발표 / 파트너 미팅 / 랜딩 페이지용 기능 설명 소스

## Feature 1: 노마드 유형 테스트

**한줄 설명:** 7개 질문으로 당신의 노마드 DNA를 진단합니다.

**상세:**
- 스토리텔링 기반 질문 — "갑자기 떠나고 싶은 생각이 들었어. 그 순간 떠오른 건?"
- 5가지 페르소나 진단: 거침없는 나그네, 어디서든 현지인, 영리한 설계자, 자유로운 영혼, 용감한 개척자
- 벡터 기반 복합 유형 (단순 다수결이 아닌 비율 계산)
- 결과가 이후 도시 추천 알고리즘에 직접 반영

**데모 포인트:**
- 질문 하나에 5~10초, 전체 소요 약 2분
- 뒤로가기 지원 (답변 변경 가능)
- 프로그레스바로 진행률 시각화

## Feature 2: 5단계 맞춤 프로필

**한줄 설명:** 현실적인 조건을 5단계로 빠르게 파악합니다.

**상세:**
- 단계별 자동 진행 — 선택 즉시 다음 단계로 (300ms 딜레이)
- 조건부 필드 — 단기 체류 선택 시 소득 대신 총 예산 질문
- 가족 동반 선택 시 자녀 나이대, 배우자 소득 추가 질문
- 페르소나 배지 + 캐릭터 GIF 이동 애니메이션으로 진행감 부여

**입력 항목:**
1. 체류 목적 (5개 옵션)
2. 체류 기간 + 형태 (4 + 3개 옵션)
3. 소득/예산 구간 + 세금 민감도
4. 동행 유형 + 조건부 (배우자 소득, 자녀 나이)
5. 선호 지역 + 라이프스타일 (복수 선택)

## Feature 3: AI 도시 추천 엔진

**한줄 설명:** 52개 도시, 39개국 비자 데이터를 AI가 분석해 TOP 5를 선별합니다.

**상세:**
- **2단계 분석 파이프라인:**
  1. DB 기반 사전 필터링 — 소득 요건, 비자 조건 불일치 도시 제외
  2. Gemini 2.5 Flash LLM — 라이프스타일, 페르소나 매칭, 종합 점수 계산
- **구조화된 JSON 출력:** 도시명, 비자 유형, 월 생활비, 점수, 추천 근거, 경고사항
- **Context Caching:** 시스템 프롬프트 + 데이터 캐싱으로 비용 절감
- **언어 정책:** 국적 기반 응답 언어 자동 결정

**데이터 포인트 (도시당):**
- 월 생활비 (USD → 만원 변환)
- 비자 유형, 무비자 체류일, 최소 소득 요건
- 인터넷 속도, 치안 점수, 영어 점수
- 코워킹 비용, 중기 렌트 비용
- 한인 커뮤니티 규모
- 기후, 입국 팁

## Feature 4: 타로 카드 UX

**한줄 설명:** 운명적 도시와의 만남을 타로 카드로 연출합니다.

**상세:**
- 5장 카드 배치 → 3장 선택 → 순차 플립 공개
- **카드 뒷면:** 나침반 로즈 SVG + "NNAI" 브랜딩
- **선택 이펙트:** 골드 보더 + 24px 글로우 + 48px 확산 그림자
- **540도 플립:** 지수적 가속/감속 (0→180→360→450→540, 1.1초)
- **순차 공개:** 0.3초 대기 → 첫 카드 → 1.8초 → 두번째 → 1.8초 → 세번째 → 2초 → 완료
- **카드 앞면:** 국기 + 도시명(한/영) + 월 비용/비자/인터넷 메트릭
- **미선택 카드:** opacity 15% + 잠금 아이콘 오버레이

**데모 포인트:**
- 데스크톱: 5장 가로 일렬 배치
- 카드 선택의 게이미피케이션 요소 (사용자가 직접 고르는 재미)
- 플립 애니메이션이 감성적 서프라이즈 제공

## Feature 5: 도시 상세 정보

**한줄 설명:** 카드를 탭하면 비자, 생활비, 인프라를 한눈에 확인합니다.

**상세:**
- **라이트박스 팝업:** 블러 배경 + 스케일 애니메이션
- **핵심 지표:** 월 생활비(만원), 비자 유형, 인터넷 속도
- **비자 상세:** 비자 유형, 체류 기간(개월), 갱신 가능 여부
- **점수:** 치안(10점), 영어(10점)
- **AI 인사이트:** 도시에 대한 AI 생성 한줄 코멘트
- **외부 링크:** 비자 공식 사이트, Flatio 숙소, Anyplace, 밋업
- **데이터 기준일:** Numbeo, NomadList 기반 검증 날짜

## Feature 6: Google OAuth 인증 + 핀 저장

**한줄 설명:** Google 계정으로 로그인하고, 마음에 드는 도시를 저장하세요.

**상세:**
- Google OAuth 2.0 기반 간편 로그인
- 관심 도시 핀 CRUD (생성/조회/수정/삭제)
- 커뮤니티 핀 — 다른 노마드들이 저장한 인기 도시 확인 (인증 불필요)
- dev 페이지에서 총 방문 수, 커뮤니티 핀 도시 수, 총 저장 핀 수 대시보드

## Feature 7: 파트너 광고 모듈 (프로토타입)

**한줄 설명:** 여행/숙소/브랜드 파트너 광고를 네이티브하게 노출합니다.

**상세:**
- 5개 브랜드 프로토타입 (Korean Air, Airbnb, Starbucks, Apple, Hostelworld)
- 사이드바(세로) / 섹션(그리드) 2가지 variant
- 픽셀아트 마스코트 (Grace, Rocky) 양쪽 배치
- 미니멀 디자인 — 120px 폭, 브랜드명 + 링크
