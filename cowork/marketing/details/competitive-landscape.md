# NomadNavigator AI - 경쟁 환경 분석

---

## 1. 시장 포지셔닝 매트릭스

```
                    높은 개인화
                        │
                        │
         NNAI ●         │
        (AI + 데이터     │
         + 감성 UX)     │
                        │
 데이터 중심 ───────────┼─────────── 콘텐츠 중심
                        │
         NomadList ●    │         ● 노마드 블로그/카페

                        │
                    낮은 개인화
```

## 2. 경쟁사 상세 비교

### NomadList
- **강점:** 가장 큰 노마드 커뮤니티, 1,000+ 도시 데이터, 실시간 코워킹/숙소 정보
- **약점:** 영어 전용, 한국 여권 기준 비자 정보 부족, 데이터만 나열 (추천 없음), 유료 ($99/yr)
- **NNAI 차별점:** 한국인 맞춤, AI 개인화 추천, 무료, 감성 UX

### Numbeo
- **강점:** 생활비 데이터 정확도 높음, 도시 간 비교 기능
- **약점:** UI 구식, 비자 정보 없음, 추천 기능 없음
- **NNAI 차별점:** 비자 통합, AI 추천, 현대적 UX

### Expatistan
- **강점:** 도시 간 생활비 1:1 비교
- **약점:** 비자/커뮤니티 정보 없음, 추천 없음
- **NNAI 차별점:** 프로필 기반 추천, 원스톱 가이드

### ChatGPT / Claude 직접 질문
- **강점:** 범용 AI, 자유로운 질문 가능
- **약점:** 구조화된 데이터 없음, 비자 정보 부정확 가능, UX 없음
- **NNAI 차별점:** 검증된 전문 데이터, 타로 카드 UX, 시각적 결과

### 한국 노마드 커뮤니티 (블로그, 카페)
- **강점:** 한국어, 생생한 후기, 커뮤니티
- **약점:** 정보 파편화, 최신성 보장 불가, 주관적
- **NNAI 차별점:** 구조화된 데이터, AI 분석, 실시간 업데이트

## 3. NNAI 핵심 경쟁력

### 3.1 한국인 맞춤 (Korean-first)
- 한국 여권 기준 비자 정보 (39개국)
- 한국어 UI/UX 최적화
- 원화 기준 생활비 표시
- 한인 커뮤니티 규모 정보
- 한국-현지국 이중과세 방지 조약 정보

### 3.2 AI 개인화
- 소득, 체류 목적, 동행 형태, 라이프스타일 종합 분석
- 5가지 페르소나 기반 추천 가중치
- DB 사전 필터링 + LLM 정밀 분석 2단계 구조
- 비자 요건 자동 매칭 (최소 소득, 체류 기간)

### 3.3 감성 UX
- 타로 카드 메타포: 운명적 도시 만남
- 540도 플립 애니메이션
- 7문항 스토리텔링 퀴즈
- 페르소나 캐릭터 (픽셀아트 GIF)

### 3.4 원스톱 가이드
- 비자 + 생활비 + 숙소 + 세금 + 커뮤니티 통합
- 외부 서비스 딥링크 (Flatio, Anyplace, Meetup)
- 입국 팁 (왕복 항공권 요구, 비자런 옵션)

## 4. SWOT 분석

### Strengths (강점)
- 한국 시장 독점적 포지션 (한국인 전용 AI 노마드 서비스 없음)
- 52개 도시 + 39개국 비자 구조화 데이터
- 타로 카드 UX — 차별화된 브랜드 경험
- 빠른 기술 스택 (Next.js + FastAPI + Gemini)

### Weaknesses (약점)
- 데이터 커버리지 한정 (52개 도시 / 글로벌 서비스 대비 적음)
- Step 2 상세 가이드 UI 미완성
- 모바일 최적화 미완성
- 사용자 리텐션 메커니즘 부족 (일회성 사용 위험)

### Opportunities (기회)
- 한국 원격근무 시장 급성장
- 각국 디지털 노마드 비자 신설 증가 추세
- 일본/대만 등 동아시아 확장 가능
- 파트너 광고 수익 모델

### Threats (위협)
- NomadList 한국어 버전 출시 가능성
- ChatGPT 등 범용 AI 정확도 향상
- 비자 정보 변경 속도 (데이터 유지보수 부담)
- 경쟁사의 AI 기능 탑재

## 5. 시장 기회

### 5.1 TAM/SAM/SOM

| 범위 | 규모 | 근거 |
|------|------|------|
| TAM | 글로벌 디지털 노마드 약 3,500만명 | MBO Partners 2025 리포트 |
| SAM | 한국인 해외 장기체류 희망자 약 20만명/년 | 외교부 해외체류 신고 |
| SOM | 1차 타겟 (IT 원격근무자 25~40세) 약 5만명 | 예상 도달 가능 |

### 5.2 성장 동인
- 한국 기업의 원격근무 정책 확대
- K-디지털 노마드 비자 인지도 상승
- 해외 체류 정보 수요 증가 (유튜브, SNS)
- 은퇴 후 해외 거주 트렌드
