      │       /  "나 이거다!"      /

## 핵심 전환 포인트 (웹 데스크톱)

| 단계 | 전환 행동 | 감정 트리거 | 이탈 위험 |
|------|----------|------------|----------|
| 랜딩 → 퀴즈 | "내 유형 찾아보기" 클릭 | 호기심 | 낮음 (CTA 명확) |
| 퀴즈 진행 | 7문항 응답 | 몰입 (스토리텔링) | 중간 (문항 수) |
| 퀴즈 → 폼 | 페르소나 결과 확인 후 진행 | 자기 발견의 기쁨 | 낮음 |
| 폼 진행 | 5단계 입력 | 기대감 축적 | 높음 (정보 입력 부담) |
| 폼 → 로딩 | "도시 추천 받기" | 기대 최고조 | 낮음 |
| 로딩 | AI 분석 대기 | 불안+기대 | 중간 (대기 시간) |
| 카드 선택 | 3장 선택 | 재미 (게이미피케이션) | 낮음 |
| 카드 플립 | 순차 공개 | 서프라이즈! | 없음 |
| 결과 → 상세 | 카드 탭 | 실용적 관심 | 낮음 |
| 상세 → 가이드 | "전체 가이드" 클릭 | 깊은 관심 | 높음 (미오픈) |
