## 1. 서비스 개요

**NomadNavigator AI (NNAI)**는 AI 기반 디지털 노마드 이민 설계 서비스다.
한국인 디지털 노마드 예비 이민자를 대상으로, Gemini 2.5 Flash LLM이 사용자 프로필을 분석해 최적의 거주 도시 TOP 3을 추천하고, 비자/예산/세금 상세 가이드를 제공한다.

**핵심 가치:**
- "나는 어떤 노마드일까?" — 유형 테스트를 통한 자기 발견
- AI가 52개 도시, 34개국, 39개 비자 데이터를 분석해 맞춤 추천
- 타로 카드 UI로 도시 추천 결과를 감성적으로 전달

## 2. 사용자 플로우 (웹 데스크톱)

```
[랜딩 페이지]
     │
     ├── "내 유형 찾아보기" ──→ [퀴즈 7문항] ──→ [페르소나 결과]
     │                                                 │
     │                                                 ▼
     └── "도시 추천 바로 받기" ──→ [폼 5단계] ──→ [타로 카드 결과]
                                                      │
                                                 카드 3장 선택
                                                      │
                                                 순차 카드 플립
                                                      │
                                                 도시 상세 라이트박스
```

### 2.1 랜딩 페이지 (`/`)
- 지구본 GIF 애니메이션 + "나는 어떤 노마드일까?" 헤드라인
- CTA 2개: "내 유형 찾아보기" (primary), "도시 추천 바로 받기" (secondary)
- "카드 플로우 미리보기" 데모 버튼 (개발용)
- 디자인: 다크 배경, fadeUp 애니메이션, 최대 폭 max-w-sm

### 2.2 노마드 유형 퀴즈 (`/onboarding/quiz`)
- 7개 문항, 5지선다
- 질문 스타일: 감성적, 스토리텔링 ("갑자기 떠나고 싶은 생각이 들었어...")
- 프로그레스바 + "이전" 뒤로가기 지원
- 완료 후 → 페르소나 결과 페이지로 이동

### 2.3 5가지 페르소나
| 페르소나 | 한국어 | 영어 | 성격 |
|----------|--------|------|------|
| wanderer | 거침없는 나그네 | The Boundless Wanderer | 끊임없이 이동, 새로운 도시 탐색 |
| local | 어디서든 현지인 | The Everywhere Local | 현지 커뮤니티 융화, 관계 중심 |
| planner | 영리한 설계자 | The Clever Architect | 합리적 판단, 인프라 중심 |
| free_spirit | 자유로운 영혼 | The Free Spirit | 계획 없는 하루, 소소한 행복 |
| pioneer | 용감한 개척자 | The Bold Pioneer | 영주권 목표, 정착 의지 |

### 2.4 프로필 입력 폼 (`/onboarding/form`)
5단계 스텝 폼, 조건부 필드 표시:

| 단계 | 질문 | 옵션 |
|------|------|------|
| 1 | 체류 목적 | 원격 근무, 프리랜서, 온라인 비즈니스, 장기 여행, 은퇴 |
| 2 | 체류 기간 + 형태 | 1~3개월/6개월/1년/영주권 + 정착형/순환형/이동형 |
| 3 | 소득/예산 + 세금 | 구간 선택 (200만원 단위) + 세금 혜택 중요도 |
| 4 | 동행 유형 | 혼자/배우자/자녀/가족 전체 + 조건부 필드 |
| 5 | 선호 지역 + 라이프스타일 | 아시아/유럽/중남미/중동/북미 + 환경 선호 |

- 페르소나 결과가 있으면 폼 상단에 배지 표시 + 캐릭터 GIF 이동 애니메이션
- 단일 선택 필드는 자동 다음 단계 진행 (300ms 딜레이)

### 2.5 타로 카드 결과 (`/result`)
- **5장 카드 배치:** 데스크톱에서는 5장 가로 나열 (`hidden md:flex`)
- **3장 선택:** 사용자가 끌리는 카드 3장 터치/클릭
- **카드 선택 시:** 골드 보더 + 글로우 이펙트 (box-shadow)
- **카드 열기:** 540도 지수적 플립 애니메이션 (1.1초, 키프레임 5단계)
- **순차 공개:** 0.3초 → 1.8초 → 1.8초 → 2초 간격으로 3장 순차 플립
- **완료 후:** 카드 탭 → 도시 상세 라이트박스 (비자, 월 생활비, 인터넷 속도 등)

### 2.6 도시 상세 라이트박스
- 국기 이모지 + 도시명 (한국어/영어)
- 핵심 지표: 월 생활비 (만원), 비자 정보, 인터넷 속도
- 비자 상세: 비자 유형, 체류 기간, 갱신 가능 여부
- 치안/영어 점수 (10점 만점)
- 도시 인사이트 (AI 생성)
- 외부 링크: 비자 정보, 숙소 검색 (Flatio, Anyplace), 밋업

## 5. 디자인 시스템

### 5.1 컬러 팔레트
- **배경:** `#0f0f0f` (메인), `#0a0c10` (랜딩), `#08090e` (dev)
- **카드:** CSS Variable `--card` (다크 테마)
- **Primary:** amber-400 계열 (골드)
- **텍스트:** `--foreground` (밝은 회색), `--muted-foreground` (어두운 회색)
- **보더:** `--border` (미묘한 구분선)

### 5.2 타이포그래피
- **세리프:** Playfair Display (도시명, 헤딩)
- **모노:** Geist Mono (메트릭, 라벨, 데이터)
- **산세리프:** Geist (본문)

### 5.3 카드 디자인
- 세로 2:3 비율 (sm: 140x245px, md: 200x350px, lg: 260x455px)
- 뒷면: 나침반 로즈 SVG + "NNAI" 로고
- 앞면: 국기 이모지 + 도시명 + 월 비용/비자/인터넷 메트릭
- 선택 시: 골드 보더 + 글로우 (box-shadow)
- 플립: 540도 지수적 회전 (0→180→360→450→540)

### 5.4 애니메이션
- **페이지 진입:** fadeUp (opacity 0→1, y 16→0, 0.5s ease-out)
- **카드 플립:** 1.1초, 5단계 키프레임, easeIn→linear→easeOut
- **순차 공개:** 300ms→1800ms→1800ms→2000ms 간격
- **페이지 전환:** AnimatePresence mode="wait", fade in/out
