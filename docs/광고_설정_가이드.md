# nnai.app 광고 설정 가이드

## 준비물
- Google 계정 (또는 Meta 계정)
- nnai.app 도메인 소유권

---

## 방법 1: Google AdSense (권장)

### Step 1 — AdSense 가입 및 사이트 등록
1. [https://www.google.com/adsense](https://www.google.com/adsense) 접속
2. Google 계정으로 로그인 → **시작하기**
3. 웹사이트 URL에 `nnai.app` 입력
4. AdSense가 사이트 소유권 확인 코드를 제공 → Cloudflare에서 HTML 메타 태그 또는 DNS TXT 레코드로 인증
5. 검토 승인 대기 (수일~수주 소요)

### Step 2 — 광고 단위 생성
1. AdSense 대시보드 → **광고** → **광고 단위 기준** → **새 광고 단위**
2. 유형: **디스플레이 광고** 선택
3. 크기: 반응형 또는 `300×600` (세로형)
4. 이름: `nnai-sidebar` 등 구분 가능한 이름
5. 생성 후 **게시자 ID**(ca-pub-...) 와 **광고 슬롯 ID** 복사

### Step 3 — data/ads_config.json 업데이트
```json
{
  "google_adsense": {
    "publisher_id": "ca-pub-여기에_본인_ID",
    "ad_slot_vertical": "여기에_슬롯_ID"
  },
  "enabled": true,
  "width_px": 300,
  "height_px": 600
}
```

### Step 4 — 배포
```bash
git add data/ads_config.json
git commit -m "feat: activate Google AdSense"
git push
```
→ Railway 자동 배포 완료 후 nnai.app 우측 사이드바에 광고 노출

## 방법 2: Meta (Facebook) Audience Network

> AdSense보다 설정이 복잡하고 모바일 앱 중심이라 웹 사이트엔 AdSense 추천

1. [https://www.facebook.com/audiencenetwork](https://www.facebook.com/audiencenetwork) 접속
2. 앱/웹 등록 → 광고 단위 생성
3. 발급받은 스크립트 코드를 `ui/layout.py`의 `_create_ad_sidebar_html()` 함수 내 반환값에 직접 붙여넣기

## 현재 구현 상태

| 항목 | 상태 |
|------|------|
| 우측 사이드바 레이아웃 | ✅ 완료 |
| 데스크톱(1024px+) 표시 | ✅ 완료 |
| 모바일(1023px 이하) 숨김 | ✅ 완료 |
| 광고 설정 파일 (`data/ads_config.json`) | ✅ 완료 (Publisher ID 교체 필요) |
| AdSense 승인 | ⏳ 미완료 (가입 필요) |

---

## 승인 전 미리보기

`data/ads_config.json`의 `publisher_id`에 `xxxxxxxx`가 포함되어 있으면
사이드바에 **"광고 영역"** 플레이스홀더가 표시됩니다. 정상 동작입니다.
